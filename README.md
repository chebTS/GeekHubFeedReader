Глюки: </b>

2) При первом старте нету спинера (индикатора загрузки), поэтому нужно подождать</b>

Реализовано:</b>
1) Сервис живет все время, раз в час проверяет обновление по ETag</b>
2) БродкастРесивер, при поднятии ВайФая, если приложение запушенно - посылает запрос на проверку обновлений </b>
3) База данных, добавление уникальны. Лайкнутые статьи можно просматривать отдельно как и было в задании.</b>
4) На девайсах с размерами large и xlarge в landscape показывается два фрагмента </b>
5) Фейсбук, регистрация и пост на стену.</b>
6) Нотификация, если приложение запущенно, а сервис нашел обновления.</b>
7) Если нет ВайФая - сообщение "No connection"</b>
8) АкшенбарШерлок, на Андроид 2.2 приложение работает.</b>
